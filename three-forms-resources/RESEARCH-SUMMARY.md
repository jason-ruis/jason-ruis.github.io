## Technical Notes

### CSV Structure

The research data is compiled into `three-forms-resources.csv` with the following structure:

```
- id: Unique identifier (kebab-case)
- title: Full resource title
- authors: Semicolon-separated author list
- resource_type: book, commentary, study_guide, sermon_series, etc.
- form: Heidelberg, Belgic, Canons, General
- summary: 1-2 sentence description
- tags: Pipe-separated keywords
- language: ISO language code
- date: Publication date (YYYY or YYYY-MM-DD)
- license_or_rights: public domain, copyrighted, CC-BY, etc.
- url: Primary URL
- canonical_url: Authoritative URL
- purchase_store: Amazon, Christianbook, etc.
- purchase_url: Direct purchase link
- asin: Amazon Standard Identification Number
- amazon_affiliate_ready: TRUE/FALSE
- recommended_for: Semicolon-separated audience
- estimated_read_time_minutes: Reading time estimate
- media_embed_url: YouTube/Vimeo embed URL
- image_url: Cover image URL
- notes: Editorial notes
- source_citation_site: Research source
- source_citation_author: Source author
- source_citation_date: Source date
- source_citation_url: Source URL
- confidence_score: 0-100 quality score
- duplicate_of: ID of canonical entry if duplicate
```

- **Accuracy**: All URLs and ASINs verified
